서버 하드웨어에 관심이 많아 사설서버를 운영했습니다. 2000년대 중반, 국내에 idc가 막 화성화될 시기에 나만의 1u서버를 가지고 싶었습니다. 당시에는 aws같은 서비스가 없었고, 온전히 운영체제를 컨트롤하려면 서버단위의 호스팅을 해야만 했습니다. 1u만이 월세(이용료)를 가장 적게 낼수 있었지만, 1u를 구성하는 컴퓨터 부품들은 대부분 서버용(xenon같은)이였기 때문에 가격이 매우 비쌌습니다. 돈이 아까웠던 저는 대만제 첸브로샷시(케이스)를 주문하고 미니보드에 core2duo를 올린 후에, 직접 쿨러를 깍아서 달았습니다. (시판되는 기본쿨러나 사제쿨러를 장착하면 1u높이가 넘어갔기 때문에 적당한 힛트싱크를 구해서 을지로 공구상에 있는 밀링머신집에서 싸인펜으로 마킹한대로 깍아달라고 부탁을 했었습니다.) 그렇게 만든 서버를 분당에 있는 idc에 월세 5만원으로 계약하고 2년여간 운영했습니다. 이외에도 개인 컴퓨터등을 같이 활용해서 freebsd, debian, solaris, redhat, nt등 여러 운영체를 설치하고 테스트했습니다.
