JAVASCRIPT 자바스크립트는 제가 가장 좋아하고 즐기는 언어입니다. 오랫동안 소프트웨어 개발자 일을 하면서, 보안에 관련없는, 서버에 부하가 가는, 사용자에게 좀더 낳은 경험을 주는 여러가지 상황들을 생각하면서, 자바스크립트로 동적 화면을 만드는것을 즐겼습니다. 요즘(2020년)에는 퍼블리셔가 스크립트를 작성하는 경우가 많은데, 퍼블리셔에게 제공받은 무료라이브러리나 오랫동안 가지고 다닌 스크립트를 아무런 스트레스 없이 고쳐서 사용할 수 있습니다. 처음 일을 시작할 당시부터 드래깅, jquery, jqgrid, 비동기(ajax), jqueryui등을 사용하고 자바스크립트에 코딩하는 시간을 매우 많이 할애했습니다. 요즘 유행하는 vue.js, node,js등도 관심이 있고 짧은 시간안에 능숙한 코딩을 할 수 있다고 자신합니다.
